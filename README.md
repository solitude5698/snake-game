# 🐍 贪吃蛇游戏

基于 C++ 实现的控制台贪吃蛇游戏，包含多种食物类型、等级系统、无敌状态等创新功能。

## ✨ 特色功能

- 🎮 5种食物类型：普通、加速、减速、双倍积分、无敌
- 🏆 等级系统：每吃5个食物升1级
- 🛡️ 无敌状态：可穿墙，持续5秒
- 💾 最高分自动保存
- 🎨 彩色控制台界面
- ⏸️ 暂停/继续功能

## 🎯 游戏控制

| 按键 | 功能 |
|------|------|
| ↑↓←→ | 控制蛇移动 |
| P | 暂停/继续 |
| H | 显示帮助 |
| R | 重新开始 |
| ESC | 退出游戏 |

## 🚀 运行方式

### 方式一：直接运行
双击 `snake_game.exe` 即可开始游戏
